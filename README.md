A React Native App which is imitate Messenger App of old version before 200.0.

Because the Messenger App has a huge change after the 200.0 versionï¼Œso I will start a new project to imitate the latest version of Messenger.
If you want to know the new Messenger app project,welcome to the new project.

## Installation

```
git clone https://github.com/tykdn/Messenger.git`
```
with yarn
```
yarn
```
or with npm:
```
npm install
```
```
npm link
```

## Usage

In the project directory:<br/>

run the app
```
npm start
```
format code
```
npm run prettier
```
build android apk
```
npm run build_android
```
